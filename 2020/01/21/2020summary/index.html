2019年度总结——暨2020年度计划

发表于2020-01-21更新于2021-01-12阅读时长1分

我身上明明没有金属制品，过安检的时候却响了，原来是因为我钢铁般的意志！

  • 岁末年初，祝爱我的人以及我爱的人们，新年快乐，身体健康，平平安安。

总结 2019

学习方面

  • 2019 里头，对于专业技能的学习方面，有学习增进，只是学习进度已经没有之前的速度那么快了，但是技能树比以往更深了。
    • 开始学习 JVM 虚拟机；
    • Spring Boot 的自动装配机制；
    • 写代码更加注重 SQL 的执行效率了，经常会用 EXPLAIN 去调试 SQL；
  • 今年还参加了一场软件水平考试，但是考试没有通过。

2020 计划

立年度目标

专业技能树方面的目标：

  • 重启 ARTS 计划；
  • Docker 集群；
  • 学习微服务的架构思维；
  • 习惯上使用 xmind 软件作图

非专业技能方面的目标：

  • 保持克制；
  • 学会剪辑视频；
  • 学会在脑力快速思考出思维地图或者思维架构；
  • 不停止思考；
